## ✨ Features

### 🛍 Product Management
- ✅ Add/edit/delete products
- 🏷 Categorize jewelry items
- 💰 Set prices, discounts, and taxes
- 📸 Product image uploads
- 🔍 Advanced search & filtering

### 🧾 Billing System
- 🧾 Invoice generation
- 🧮 Automatic tax calculations
- 💸 Discount applications
- 🖨 Bill printing support
